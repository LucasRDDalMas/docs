# Documentation

## Purpose

> Maintaining this area means: all active projects have up-to-date notes, decisions are recorded,
> and new team members can onboard using existing material.

## Standards

- Every project gets a project note before work starts
- Meetings are captured within 24 hours using [[Meeting Note Template]]
- Resources are summarised before being archived
