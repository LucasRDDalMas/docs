# Portal

End-user web portal documentation. Each page maps 1-to-1 to a frontend route.

| Route | Description |
|-------|-------------|
| [/dashboard](dashboard/index) | KPI overview and activity feed |
| [/users](users/index) | User management table |
| [/users/:id](users/detail) | User profile and history |
| [/orders](orders/index) | Order list with filters |
| [/orders/:id](orders/detail) | Order detail and timeline |
| [/settings](settings/index) | Account and workspace settings |
